# Stabil visszatérési pont – Mátra Hard Trail

## Állapot

Ez a verzió a Mátra Hard Trail tiszta, stabil béta állapota.

```text
Mátra Hard Trail v0.9
stabil béta
élesben tesztelhető
```

## Mit jelent ez?

Ez az a verzió, amely:

- külön repo-ban fut,
- saját GitHub Pages linken él,
- letisztult online kitöltőfelületet ad,
- tartalmazza a döntési biztonsági indexet,
- tartalmazza az önreflexiós kérdéseket,
- tartalmazza az 1 oldalas A4 PDF összegző lapot,
- tréningben már használható.

## Fontos megjegyzés

Ez a repo egyik tudatosan megőrzött, tiszta állapota.

Ha a jövőben továbbfejlesztés történik, és az irány túlbonyolódik, elcsúszik, vagy romlik a használhatóság, ehhez a verzióhoz érdemes visszatérni.
